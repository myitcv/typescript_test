## `typescript_test`

Download and setup:

```
git clone git@github.com:myitcv/typescript_test.git typescript_test && cd $_
export PATH=$PWD/node_modules/.bin:$PATH
```

Setup:

```
npm install
jspm install
tsd install
```

Run:

```
gulp
node main.js
```

`gulp watch` can be used to recompile on the fly
